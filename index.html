Run PHP Code

Run PHP Code is an open source Mac OS X application which allows you to run PHP code.

Download Application

Don't waste your time with files, browsers or consoles for a piece of code. Just write and run your code with different PHP versions.

Features

  • Syntax highligthing (using ace editor for this).
  • Supports multiple PHP version.
  • Supports HTML output.
  • Code execution time functionality.

Requirements

OS X 10.6 or later.
